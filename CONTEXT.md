# Expert Base — ナレッジベース

プロフェッショナルの個人知識をローカル完結で蓄積・検索・編集するデスクトップアプリの文脈。ファイルが知識の真源であり、それ以外はすべて派生物である。

## Language

**ナレッジベース（KB / 知识库）**:
ユーザーが選んだルートフォルダ配下に条目と派生インデックスを持つ、知識の入れ物。同時にアクティブなのは常に 1 つ。

**条目（Entry / 条目）**:
frontmatter 付き Markdown ファイルとして保存される知識の最小単位。ファイルの内容が条目の真源。
_Avoid_: ノート、記事、ドキュメント

**条目持久化（Entry Persistence / 条目持久化）**:
条目の誕生・改写・削除が、必ず索引の鏡像更新と一体で起きるという不可分の領域動作。どの経路から書いても同じ一箇所を通る。
_Avoid_: 「ファイル書き込み+索引更新」（二つの別動作として語ること）

**派生インデックス（Derived Index / 派生索引）**:
条目ファイル群から再構築可能な検索・リンク用の索引。データの一部ではなくキャッシュであり、失われても条目からいつでも復元できる。
_Avoid_: データベース（真源であるかのような呼び方）

**確認ゲート（Confirm Gate / 确认门）**:
工作坊で AI が破壊的な操作（書き込み・上書き・削除）を実行する前に、必ずユーザーの承認を挟む関門。「いつ書くか」の主導権はユーザーにある。
